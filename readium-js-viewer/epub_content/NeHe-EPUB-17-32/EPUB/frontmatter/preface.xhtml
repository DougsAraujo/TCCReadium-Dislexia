Preface

This book is the second voume in a series exploring using WebGL and three.js using the well-known NeHe lessons as basis for the exploration. These books began as a collaboration between Johannes Raida and Ric Wright. Johannes began writing some lessons using the well known OpenGL lessons known as NeHe. He posted these to his website. Ric then joined in and began porting the rest of the lessons to three.js and WebGL.

All 48 (or all those relevant) lessons have now been ported. This volume covers the second 16 lessons (17 through 32). Not all the lessons have been ported for various reasons. Some of the lessons cover subjects (e.g. using bitmap fonts) which are unneeded or "just work" in three.js so don't warrant spending time covering.

Ric is the Director of Engineering for the Readium project, which is an open-source foundation dedicated to providing web apps and a SDK for rendering EPUB, the standard format for eBooks. As a demonstration of the capabilities of EPUB and Readium, Ric took the lessons and created this EPUB. The details of how this book was created are covered in the colophon at the end of the book.

If you have any feedback, suggestions or bug reports about this book, you are strongly encouraged to contact Ric directly.
